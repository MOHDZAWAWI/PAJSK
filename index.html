⚙️ Sambungan Database Awan Diperlukan

Tahniah! Anda berjaya menerbitkan laman web ini ke GitHub Pages. Walau bagaimanapun, untuk membolehkan semua guru menyimpan data pada pangkalan data yang sama, anda perlu memasukkan konfigurasi Firebase (Percuma) anda sendiri ke dalam fail kod.

Langkah penyediaan mudah (Kurang 5 minit):

  1. Layari Firebase Console & mulakan projek percuma.
  2. Aktifkan Firestore Database & mulakan dalam "Test Mode" atau benarkan capaian.
  3. Di tab Authentication, aktifkan log masuk Anonymous (Tanpa Nama).
  4. Tambah Web App baharu pada projek anda, dan salin nilai firebaseConfig.
  5. Buka fail kod anda di GitHub, cari pembolehubah MY_CUSTOM_FIREBASE_CONFIG di bahagian bawah fail, gantikan nilainya, dan simpan (commit)!

Sistem kini berjalan dalam mod demonstrasi luar talian sahaja.
